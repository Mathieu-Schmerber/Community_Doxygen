Description

Query parameters can be used on every specified route. Those are always optional and hold default values that are specified for each route using them.

Pagination

CommunityAPI paginates all data before sending it back to the client. Please refer to the endpoint documentations bellow to retrieve the pagination arguments.

Community API will write the pagination data when occuring, in the metadata header X-Pagination.

HTTP/1.1 200 OK
X-Pagination: {"PageDataCount":10,"PageSize":10,"TotalDataCount":20,"CurrentPage":1,"PageCount":2,"HasNext":true,"HasPrevious":false}
  • PageDataCount: Number of data present on the page
  • PageSize: PageSize argument
  • CurrentPage: Page number
  • TotalDataCount: Total number of data
  • PageCount: Total number of pages
  • HasPrevious: A page is available at CurrentPage - 1
  • HasNext: A page is available at CurrentPage + 1

Sorting

CommunityAPI sorts all data before sending it back to the client. Please refer to the endpoint documentations bellow to retrieve the sorting arguments.

You can ask CommunityAPI to sort your data by specifying a field.

‍/accounts/me/friends?SortField=username // sort friend list by username

You can also sort your data by specifying a nested field.

‍/courts/search?SortField=creator_user.username // sort courts by the username of the creator of the court

Filtering

CommunityAPI can filter data before sending it back to the client.

The Filters parameter can be used to only select data by field values.

‍/courts/search?Filters=rate_average>4 // get all courts with a rate average greater than 4/5

The available operator are =, >, <.

Multiple filters can be specified by using the ; separator.

‍/courts/search?Filters=rate_average>0;rate_average<5 // get all courts with a rate average between 1 and 4

Filters can be used to match string using wildcards *

‍/accounts/me/friends?Filters=username=Elon* // get all friends with a name beginning with Elon /accounts/me/friends?Filters=username=*Musk // get all friends with a name ending with Musk /accounts/me/friends?Filters=username=*Elon* // get all friends with a name containing Elon
